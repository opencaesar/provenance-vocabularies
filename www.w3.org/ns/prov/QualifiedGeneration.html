rdfs:comment

Generation is the completion of production of a new entity by an activity. This entity did not exist before generation and becomes available for usage after this generation. If this Activity prov:generated Entity :e, then it can qualify how it performed the Generation using prov:qualifiedGeneration [ a prov:Generation; prov:entity :e; :foo :bar ].
